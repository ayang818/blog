什么是DevOps？什么是 CI/CD 流水线？

我归纳一下，

  1. DevOps是敏捷开发的一种主流实现方式，相比于传统的瀑布模型，DevOps为我们快速开发应用，快速发布应用，快速部署应用提供了一种方法论。
  2. DevOps中大致有如下几个环节
    1. Version Control （版本控制）
    2. Continuous Integration（持续集成）
    3. Continuous Delivery （持续交付）
    4. Continuout Deployment（持续部署）
